Quick Start

Generate demo test files (EICAR test, suspicious scripts, clean files, simulated malware) into the scan_targets/ folder, then scan that folder in the Scanner tab.

How It Works
01
Signature Scan — Each file is hashed (SHA-256). The hash is compared against the known malware database. A match = infected.
02
Heuristic Scan — File contents are searched for suspicious byte patterns (cmd.exe /c, PowerShell encoding, WScript.Shell, registry run keys, etc.).
03
Quarantine — Detected files can be moved to a sandboxed quarantine folder so they cannot run or spread.
⚠
Note: This tool is for education and ethical use only. It will not detect real-world malware unless real signatures are added.
